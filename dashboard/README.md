## [Dashboard](https://hopeful-lalande-7b9499.netlify.app/#/)


It is **a client-side** application built with [VueJS](https://vuejs.org/), Vuex and TypeScript class-based components. 

The application is deployed on [Netlify](https://hopeful-lalande-7b9499.netlify.app/#/).

### Usage

Install node modules:
```bash
npm install
```

Start the project:
```bash
npm run serve
```

App running at:
```bash
 http://localhost:8080/
```
